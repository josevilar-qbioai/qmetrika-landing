# QMetrika Labs · Bio-IA

Landing page for the Bio-IA research line at QMetrika Labs.

## Live site

Hosted via GitHub Pages: `https://josevilar-qbioai.github.io/qmetrika-landing/`

## Contents

- `index.html` — Landing page (self-contained, no build step)
- `qmetrika_video_es.html` — Project overview video (Spanish)
- `qmetrika_video_en.html` — Project overview video (English)

## Setup

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: **main** / root
5. Save — the site will be live in ~1 minute

No build tools, no dependencies. Pure static HTML.
